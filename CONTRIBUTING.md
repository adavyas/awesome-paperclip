# Contributing to Awesome Paperclip

Thank you for your interest in contributing! This list is community-driven, and we appreciate every addition.

## How to Add an Entry

1. **Fork** this repository.
2. **Edit** `README.md` and add your link under the appropriate category.
3. **Submit** a pull request.

### Entry Format

Please use the following format when adding a link:

```markdown
- [project-name](https://github.com/owner/repo) - A short description of the project.
```

- Start the description with a capital letter.
- End the description with a period.
- Keep descriptions concise (one sentence).

### Which Category?

| Category | What belongs here |
| --- | --- |
| **Official** | Repositories maintained by the Paperclip team |
| **Plugins** | Extensions and integrations that add capabilities to Paperclip |
| **Tools & Utilities** | Bots, bundles, CLIs, and helper tools for the ecosystem |
| **Resources** | Guides, books, tutorials, blog posts, and videos |

If none of the existing categories fit, feel free to suggest a new one in your PR.

## Quality Standards

To keep the list useful, please ensure your submission meets the following criteria:

- **Relevant** — The project must be directly related to [Paperclip](https://github.com/paperclipai/paperclip).
- **Public** — The repository should be publicly accessible.
- **Maintained** — The project should show recent activity (commits within the last 6 months).
- **No duplicates** — Check the list first to make sure the project is not already included.
- **Working** — The project should be functional and not in a broken state.

## Pull Request Guidelines

- Use a clear PR title, e.g., `Add paperclip-plugin-xyz to Plugins`.
- One addition per pull request keeps reviews fast.
- Make sure the list is alphabetically ordered within each category.
- Verify that your links work before submitting.

## Updating Your Entry

If you need to update an existing entry (e.g., change the description or URL), follow the same process: fork, edit, and submit a PR.

## Code of Conduct

Be respectful and constructive. We are all here to make the Paperclip ecosystem better.
